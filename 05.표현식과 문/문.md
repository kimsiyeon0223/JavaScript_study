# 문 (명령문)
문은 프로그램을 구성하는 기본 단위이자 최소 실행 단위이다.   
&emsp; -> 문의 집합으로 이루어진 것이 **프로그램**, 문을 작성하고 순서에 맞게 나열하는 것이 **프로그래밍**이다. (문은 여러 토큰으로 이루어진다.)

문은 *선언문, 할당문, 조건문, 반복문* 등으로 구분할 수 있다.
```js
// 변수 선언문
var x;

// 할당문
x = 100;

// 조건문
if (x == 100) {console.log(x);}

// 함수 선언문
function hello() {}

// 반복문
for (var i = 0; i < 40; i++) {console.log(i)}
```
